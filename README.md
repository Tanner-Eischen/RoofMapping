# Roof Mapping

Project bootstrapped with Next.js 14 (TypeScript), Tailwind CSS, shadcn/ui, ESLint/Prettier, and Husky.

## Prerequisites
- Node.js 18+
- npm (or pnpm)

## Setup
1. Install dependencies: `npm install`
2. Initialize Husky hooks: `npm run prepare`
3. Start dev server: `npm run dev`

## Scripts
- `dev` – start Next.js dev server
- `build` – production build
- `start` – run production server
- `lint` – lint with ESLint
- `format` – format with Prettier
- `test` – run unit tests (Vitest)

## Notes
- Tailwind configured with custom `sage` palette.
- Basic `Button` component available under `components/ui/button.tsx`.
